Introduction

Comment cela fonctionne ?

Ce module Omeka S vous permet d’importer des notices bibliographiques depuis Koha.

Prérequis

  • Le plugin Koha OmekaExport doit être installé et actif.

  • L’API REST Koha et son accès par la méthode d’authentification “ « client credentials » (préférence système: RESTOAuth2ClientCredentials) doit être actif.

  • Le vocabulaire personnalisé Koha doit être installé.

Où est la configuration

Actuellement le profil de correspondances Standard est inclus dans le module mais vous pouvez ajouter un profil de correspondances personnalisé dans le fichier module.config.php file. (Voir aussi Configuration)
