Configuration

Ajouter vos accès koha

Dans le fichier module.config.php ajouter vos accès à l’API Koha (plugin OmekaExport nécessaire):

'koha-import' => [
    'url' => '',
    'client_id' => '',
    'client_secret' => '',
],

Ajouter votre profil de correspondances personnalisé

Pour ajouter votre propre profil de correspondances ajouter ceci à votre fichier module.config.php comme ceci:

'koha_import_mapping' => [
    'factories' => [
        'your_mapping' => Path\To\Your\Mapping::class,
    ],
];

Ajouter un bucket S3 comme source de médias stockés

Ajouter ceci dans votre fichier module.config.php:

'koha-import_bucket' => [
    'key' => '',
    'secret' => '',
    'region' => '',
    'bucket_name' => '',
];
